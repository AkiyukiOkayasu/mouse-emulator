# mouse-emulator  
A Karabiner Elements rule for emulating the behavior of the mouse with keyboard shortcuts.

# Install  
Add mouseEmulator.json to **~/.config/karabiner/complex_modifications**.  
Then, the mouse-emulator menu will automatically be added to the Karabiner Elements Preferences "complex modifications".  
So, you can add the rule from "Add rule":)

# Features  
left click: ctrl+space
